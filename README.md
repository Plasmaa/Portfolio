# My Portfolio

A personal portfolio website built with Next.js (Frontend) and Django (Backend).

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS, TypeScript
- **Backend:** Django, Django REST Framework

## Getting Started
1. **Backend:**
   ```bash
   cd backend
   pip install -r requirements.txt
   python manage.py runserver
   ```

2. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
